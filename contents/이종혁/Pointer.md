# 포인터 
> 메모리 주소를 저장하는 변수로 특정 주소의 데이터에 접근할 수 있다

 * 포인터 선언: [TYPE * 변수이름]
 * ex) int * ptr;

 ***포인터의 크기는 8바이트 고정 크기다***
 
<br>

### 주소 연산자 (&)
> 주소 연산자 `&`를 사용하면 변수에 할당된 메모리 주소를 확인할 수 있다


<br>

### 역참조 연산자(*)
> 변수의 주소를 얻는 것 자체로는 그다지 쓸모가 없다

 * 역참조 연산자 `*`를 사용하면 특정 주소의 데이터에 접근할 수 있다 

<br>

``` cpp
#include<iostream>

using namespace std;

int main(int argc, char * argv[])
{

    return 0;
}
```
